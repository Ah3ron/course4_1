# Скрипты для работы с базой данных

## Заполнение базы данных тестовыми данными

Для заполнения базы данных тестовыми данными белорусских организаций и вымышленных физических лиц выполните:

```bash
cd backend
python scripts/seed_database.py
```

Скрипт создаст:
- Оценки для 15 белорусских организаций за последние 2 года (по 24 оценки на каждую)
- Оценки для 15 физических лиц за последний год (по 12 оценок на каждого)

Все данные включают временные тренды, показывающие изменение показателей со временем.
